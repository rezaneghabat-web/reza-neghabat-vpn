# Reza Neghabat VPN Manager

این مخزن شامل سیستم مدیریت فیلترشکن شخصی شماست.

## ویژگی‌ها
- **پنل مدیریت:** مدیریت کاربران و لینک‌های اشتراک
- **آپدیت خودکار:** دریافت کانفیگ‌های جدید هر ۱۲ دقیقه
- **بدون سرور:** کاملاً رایگان و اجرا شده روی GitHub Pages

## نحوه استفاده
1. به بخش Settings > Pages بروید.
2. Source را روی `main` قرار دهید.
3. لینک سایت شما (داشبورد) فعال می‌شود.
